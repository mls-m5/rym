rym

A game about beauty, explosions, lines, squares and a triangle

Download for windows or build it for your self on linux:

cd (en mapp där du vill lägga det)
git clone git@github.com:mls-m5/rym.git
sudo apt-get install build-essential libsdl1.2-dev
cd rym
make
./rym
